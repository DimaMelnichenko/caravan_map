# Caravan Map — Проект интерактивной карты мира

## Обзор проекта

**Caravan Map** — это интерактивная 2D-карта мира в стиле фэнтези, созданная на основе игрового движка **Phaser 3**. Проект представляет собой симуляцию торговых путей между городами, где караваны автоматически перемещаются по заданным маршрутам, перевозя товары.

### Основные возможности

- 🗺️ **Интерактивная карта** с зумом и панорамированием
- 🏘️ **Города** с настраиваемой экономикой (производство/потребление товаров)
- 🚩 **Державы** с параметрами культуры, воинственности, религии
- 🐪 **Караваны** — автоматическое движение товаров по маршрутам
- 🛣️ **Торговые пути** — сухопутные и морские с редактируемой геометрией
- 🖋️ **Редактор границ** — рисование границ между державами
- 🎮 **Режим Мастера** — полный редактор всех объектов мира

## Сборка и запуск

### Установка зависимостей

```bash
npm install
```

### Запуск сервера

```bash
npm start
```

После запуска откройте `http://localhost` (порт указан в server.js, который необходимо создать или настроить).

### Миграция данных

```bash
node migrate.js
```

## Управление в игре

| Действие | Управление |
|----------|------------|
| Перемещение по карте | ЛКМ + перетаскивание / Средняя кнопка мыши |
| Зум | Прокрутка колёсика мыши |
| Выбор города | Клик по городу |
| Режим Мастера | Кнопка "🛠️ Режим Мастера" |

## Режим Мастера (Редактор)

### Создание городов
1. Включите "Режим Мастера"
2. Нажмите "🏘️ Новый город"
3. Кликните на карту для размещения

### Прокладка путей
1. Нажмите "🗺️ Проложить путь"
2. Выберите первый город
3. Выберите второй город
4. Редактируйте точки пути (синие маркеры)

### Редактирование границ
1. Нажмите "🖋️ Редактор границ"
2. Создайте новую линию или выберите существующую
3. Перетаскивайте точки для изменения формы

## Архитектура

### Игровой цикл
1. **BootScene** → Инициализация движка
2. **PreloadScene** → Загрузка текстур и звуков
3. **MainScene** → Основная логика:
   - Загрузка данных из БД через `DataService`
   - Создание городов, путей, караванов
   - Обновление позиций караванов каждый кадр
   - Обработка ввода пользователя

### Синхронизация с сервером
- `DataService` — CRUD операции для городов, путей, стран, границ
- `CaravanService` — управление состоянием караванов на сервере
- Караваны обновляют позицию через POST-запросы к API

### Хранение данных
Все данные мира хранятся в SQLite (`world.db`):
- `cities` — города (координаты, население, экономика)
- `routes` — пути (тип, точки, транспорт)
- `countries` — державы (параметры, владельцы городов)
- `borders` — границы (JSON массив точек)
- `caravans` — караваны (прогресс, груз)
- `items` — типы товаров

## Примечания

- **server.js** отсутствует в репозитории — необходимо создать Express-сервер с API endpoints:
  - `GET /api/load` — загрузка всех данных мира
  - `POST /api/cities`, `DELETE /api/cities/:id`
  - `POST /api/routes`, `DELETE /api/routes/:id`
  - `POST /api/countries`, `DELETE /api/countries/:id`
  - `POST /api/borders`, `DELETE /api/borders/:id`
  - `GET /api/caravans`, `POST /api/caravans/:id/update`
  - `POST /api/routes/:id/init-caravans`

- Для работы требуется карта-текстура `assets/images/map.png` (или аналогичное имя)
